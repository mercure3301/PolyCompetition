# Scav Shooter

Un jeu de space shooter développé en JavaScript/Canvas

Lien du jeu : [https://spacewar-scav.vercel.app/](https://spacewar-scav.vercel.app/)

## Contrôles

- **Flèches directionnelles** ou **WASD** : Déplacer le vaisseau
- **Espace** ou **Clic souris** : Tirer

## Règles du Jeu

### Objectif

Survivez le plus longtemps possible en éliminant les ennemis et le boss pour obtenir le meilleur score.

### Mécaniques

1. **Système de munition limité**
   - Vous commencez avec 50 munitions
   - Chaque tir consomme 1 munition
   - Quand vous n'avez plus de munitions, la partie prend fin

2. **Pick-ups**
   - Il y a des drops qui apparaissent aléatoirement et qui donnent plus de munitions
   - Il y a aussi des pick up de vie qui apparaissent parfois

## Installation et Lancement

1. Téléchargez tous les fichiers du projet
2. Ouvrez `index.html` dans un navigateur web
3. Cliquez sur "Commencer" pour démarrer le jeu
